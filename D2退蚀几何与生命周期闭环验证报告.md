## 2. 退蚀几何

### 2.1 圆柱簇

1. 每段侧壁使用点值 `z` 掩膜读取局部平均温度，解析侧面积为 `2*pi*r_i*L_i`；掩膜面积比超界直接失败，不使用温度回退。
2. 相邻半径不等时，内肩面面积为 `pi*abs(r_i^2-r_j^2)`，按相邻侧壁平均温度计算升华，并归属较大半径段。
3. 侧壁与肩面的表面体积损失按 `dV/dr_i=2*pi*r_i*L_i` 守恒映射为逐段半径变化。
4. 两个接触电极外端面不辐射、不升华；内部肩面参与辐射和质量损失。
5. 肩面采用体积等效的半径降阶映射，并非显式轴向移动。这一假设必须在最终报告中说明；平滑轮廓可在后续设计中消除该近似。

### 2.2 Zigzag 簇

1. 每个非零路径段保留独立连续边长，不再用最小边长统一重建。
2. 每个 90 度转角使用相邻两段最大边长的 connector cube，避免不等边长 cuboid 在转角产生非流形微小台阶。
3. 直段只统计 connector 之外的真实暴露长度；短端段按实际端点残余区间采样，不再假设段中心一定暴露。
4. 每个 connector 的四个外侧面面积为 `4*L^2`，平均分配给相邻两段；尺寸差肩面归属较大段。等边、等温时该分配保持各段均匀退蚀。
5. 两端 stub 独立保存半径状态；侧壁和与方截面连接处的暴露内肩面均造成质量损失，接触电极外端面排除。
6. COMSOL 重建边长按初始边长的 `1%` 量化以稳定网格；连续退蚀状态、体积状态和 20% 失效时刻保持精确，最大投影误差上界为 `0.5%`。该量化尺度转入 D6 收敛验证。

## 3. 生命周期状态机

每个宏步统一执行：

```text
由已验证端点计算退蚀率
-> 取时间步、20% 交点和寿命 cap 的最小值
-> 构造候选几何并求解 COMSOL
-> 插值检查步间超温
-> 只对有效时间区间做梯形积分
-> 成功后提交时间、几何、功率和能量状态
```

终止语义：

| 状态 | 含义 |
|---|---|
| `OK` | 某个独立特征精确达到 20%，`lifetimeExact=true` |
| `CENSORED_LIFETIME_CAP` | 到达仿真寿命上限，不能报告真实寿命 |
| `CENSORED_STEP_LIMIT` | 到达宏步上限，不能报告真实寿命 |
| `CENSORED_NEGLIGIBLE_EROSION` | 退蚀可忽略，功率积分到 cap，但寿命仍为删失 |
| `FAIL_OVERTEMP_DURING_EROSION` | 步内插值达到 3273.15 K，只积分到超温交点 |
| `FAIL_EROSION_SOLVE` | 候选端点求解失败，不提交该端点 |

内部重试只移除并重建当前模型，不在同一 Python 进程中停止/重启 `mph` 客户端；真正断连交给外层断点机制重启同一点。

## 6. 排障记录

1. 直接连接不等边长 cuboid 会在转角形成微米级非流形肩面，COMSOL 出现温度自由度 NaN。
2. 增加 max-side connector 后拓扑稳定，但小于网格分辨率的连续边长差仍会导致求解失败；1% 几何量化后初始和退蚀几何均可重复求解。
3. 旧 Box `intersects` 选择会把 Union 转角内部重叠面计入多段侧壁，曾造成 36 秒约 `1.78%` 的虚假损失。
4. 改为真实暴露区间的点值掩膜后，同一 36 秒案例最大损失为 `0.01564%`，掩膜面积比接近 1，百倍放大问题消失。

失败尝试及最终通过日志均保留在 `tests/`，用于说明修正依据，不作为正式性能数据。
