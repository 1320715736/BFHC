## 2. 已确认口径

### 2.1 几何与电极

- 初始坯料：直径 5 mm、高度 15 mm 的钨圆柱。
- 两端距离：最终拓扑体两端严格保持 15 mm。
- 材料体积：总材料体积不变，可重分布到原始圆柱外部。
- 电极：直径 5 mm 圆形，电极相对位置不变；优化件必须与电极接触，连接处可粗可细。
- 接触电极的两个端面：不考虑向外辐射和升华。
- 结构约束：保持整体性，避免孤立碎片；特征尺度由团队按构型自行定义。

### 2.2 电压与额定工况

- 100 V 是仿真电压上限，不是固定必须施加的工作电压。
- 每个固定几何应寻找不超过 100 V、且满足温度和寿命权衡的额定工况。
- 初始圆柱不能直接按 100 V 稳态结果作为基线，应按额定工况计算。
- 冷态电阻低导致的升温过流问题可忽略；动态升温轨迹属于加分/补充项。

### 2.3 热边界与辐射

- 外接球：视为 0 K、发射率 1 的统计面，用于统计有效辐射。
- 有效辐射波段：只统计 0-3 um，波段以外不计入有效辐射。
- 对流：不使用对流散热。
- 电极导热：主口径选用固定温度边界，即铜电极为室温，忽略接触热阻/接触电阻。
- 无穷远边界作为备选/敏感性分析口径，不进入主流程。
- 端面：接触电极两端面不辐射、不升华；但端面电极导热边界仍需建模或说明。

### 2.4 温度约束与均匀性

- 温度硬约束：稳态和瞬态温度均小于 3000 C，即 3273.15 K。
- 退蚀全过程也必须检查温度约束，不能只检查初始额定稳态。
- 温度均匀性评分公式：

```text
U = (Tmax - Tmin) / Tmean * 100%
```

- 温度均匀性统计时刻：最终优化后器件达到稳态温度时刻。
- 统计域：全材料固体域；`Tmean` 采用体积平均温度。
- 分档：`U < 5%` 得 5 分，`5%-10%` 得 3 分，`>10%` 得 0 分。

### 2.5 寿命与有效辐射

- 失效准则：任意自定义特征尺度因升华/蒸发变化达到 20% 时失效。
- 寿命评分公式文件给出：

```text
eta_L = (L_opt - L_ini) / L_ini * 100%
```

- 评分文字中的“初始寿命的 50%/40%-50%/30%-40%”与提升率公式存在潜在冲突；当前同时记录：

```text
R_L   = L_opt / L_ini * 100%
eta_L = (L_opt - L_ini) / L_ini * 100%
```

- 有效辐射总量提升公式：

```text
eta_E = (E_opt - E_ini) / E_ini * 100%
```

- `E` 的最终时间口径尚未完全确认；当前同时保留：

```text
initialP03sphere_W      初始额定稳态 0-3 um 外接球净辐射功率
lifeAvgP03sphere_W      生命周期平均 0-3 um 外接球净辐射功率
lifeTotalP03sphere_J    生命周期累计 0-3 um 有效辐射能量候选项
```

## 8.待解决的问题（已经基本解决，都输出，得到答复后再选择具体口径）

有，而且有几处会直接影响模型和报告口径。主要是这 6 个：

1. **寿命评分公式有冲突**

官方公式文件写：

```
eta_L = (L_opt - L_ini) / L_ini * 100%
```

但分段标准写的是：

```
优化设计寿命 >= 初始寿命的 50%、40%-50%、30%-40%
```

这两个不是同一个口径。前者是“寿命提升率”，后者是“寿命保持率”。结合原赛题“优化后寿命最低不得低于初始寿命 30%”，我判断评分实际更可能按：

```
R_L = L_opt / L_ini * 100%
```

建议报告里同时列 `R_L` 和 `eta_L`，内部优化先按 `R_L >= 50%` 争取满分档。

1. **“有效辐射总量”到底是功率还是生命周期总能量不完全明确**

新公式写：

```
eta_E = (E_opt - E_ini) / E_ini * 100%
```

但原赛题同时提过：

- 初始时刻 0-3 um 净辐射功率
- 生命周期平均辐射功率
- 有效辐射总量提升

这里 `E` 可能被理解为生命周期累计有效辐射能量，也可能被粗略当作有效辐射功率指标。

建议我们后续同时输出三项：

- `initialP03sphere_W`
- `lifeAvgP03sphere_W`
- `lifeTotalP03_J = lifeAvgP03sphere_W * lifetime_s`

报告主指标用“生命周期有效辐射总量”，同时保留功率指标解释。

1. **温度均匀性的统计域不确定**

公式明确：

```
U = (Tmax - Tmin) / Tmean * 100%
```

并写“全域温度统计报表”。

但企业答疑又说接触电极的两个端面不考虑辐射和升华。如果电极端面附近有固定温度或强边界效应，`Tmin` 可能被端面区域主导，导致 `U` 很大，未必反映发热自由表面的均匀性。

建议同时输出两套：

- 全材料域：`U_volume_all`
- 有效工作域/自由表面：`U_active_surface` 或 `U_free_surface`

报告主表采用“全域”口径，附表解释有效自由表面口径。

1. **100V 是“固定工况”还是“上限”存在历史冲突**

原赛题多处写“给定 100V”或“施加规定最大直流电压 100V”。

Excel 答疑明确说：`100V 是仿真计算中的电压使用上限`，每个几何应寻找自己的额定工况，不能超过 100V，也不能超温。

建议模型主流程继续用：

```
Vwork = min(100V, 满足 Tmax < 3273.15K 的额定电压)
```

但复赛报告单独补一个“100V 上限工况说明”：若直接 100V 超温，则判定不可作为稳态额定工况，只作为上限约束或瞬态风险分析。

1. **外接球温度口径有冲突**

原赛题描述环境是“室温黑体”；Excel 答疑说外接球可认为是 `0K、发射率 1` 的面，用于统计有效辐射。

现有代码主要按 `Tamb = 293.15K` 处理。两者对 3000K 附近高温辐射影响不算巨大，但口径上必须统一。

建议复赛前做一次对比试算：`Tamb=293.15K` vs `Tamb=0K`。报告主口径优先采用答疑的 `0K 外接球统计面`，同时说明室温项影响很小。

1. **端面“不辐射、不升华”和现有代码实现冲突**

这是明确冲突，不是不确定。

现有 runner 里辐射积分多处是 `selection().all()`，这会把接触电极的两个端面也计入辐射统计；退蚀/升华如果按全表面或全段平均，也可能间接包含端面效应。

这需要改代码：建立自由表面选择集，辐射功率积分和升华寿命计算都排除两个接触电极端面。

结论：最需要先定的是 **寿命评分口径、有效辐射总量口径、温度均匀性统计域、外接球温度**。其中端面排除不是讨论项，是必须修正项。
