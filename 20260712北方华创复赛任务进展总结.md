## 2. 复赛口径基线

当前采用的主口径如下：

| 项目 | 当前口径 |
|---|---|
| 初始坯料 | 直径 `5 mm`、高度 `15 mm` 钨圆柱 |
| 体积 | 总材料体积守恒 |
| 电压 | `100 V` 是上限，不是固定工作电压 |
| 额定工况 | 每个几何寻找不超过 `100 V` 且满足温度约束的工作电压 |
| 电极热边界 | 固定室温铜电极主口径 |
| 端面 | 接触电极两端面不向外辐射、不升华 |
| 有效辐射 | 只统计 `0-3 um`，外接球按 `0 K`、发射率 `1` 统计面 |
| 温度约束 | 全过程 `Tmax < 3273.15 K` |
| 温度均匀性 | `U = (Tmax - Tmin) / Tmean * 100%` |
| 寿命失效 | 任意自定义特征尺度退蚀/升华变化达到 `20%` |
| 有效总量 | 主用 `lifeTotalP03sphere_J`，同时保留初始功率和生命周期平均功率 |

仍需在报告中同时列出 `R_L` 和 `eta_L`，因为官方寿命公式与分档文字存在口径冲突。
