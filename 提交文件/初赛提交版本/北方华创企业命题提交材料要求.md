# 十七、“北方华创”企业命题

| 提交文件夹 | 材料 | 内容要求 |
| --- | --- | --- |
| 技术文档 | 技术文档 | 根据赛题内容调研相关资料，形成调研报告。结合调研、对赛题的理解及问题要求，给出解决方案并介绍关键技术（包括但不限于问题理解、物理模型、算法设计、求解结果、算法性能分析、创新点、待改进项等）。 |
| PPT | PPT | 内容同上述技术文档，以PPT图文结合的形式展现作品重点信息，具体细节体现在技术文档内即可，总结报告无需过多赘述。 |
| 技术数据（代码类） | 技术数据总压缩包 | ZIP或RAR压缩包格式，打包提交项目仿真模型与算法源代码，内容包括但不限于提交完整的、可复现的强化学习训练代码与仿真驱动代码、优化过程中的关键日志、性能指标记录文件、最终优化部件的三维模型文件（STEP及STL格式）。 |
| 演示视频 | 演示视频 | MP4格式，清晰度1080P，优先横屏录制，视频时长建议控制在5分钟内。展示系统运行流程：从启动训练到最终设计生成、动态演示拓扑演化过程与伴随的性能（温度均匀性、应力等）变化曲线、对最终优化设计进行多角度渲染展示，并解说其结构特点与性能优势。 |
