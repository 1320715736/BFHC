# 优化方法选择论证：Optuna TPE 贝叶斯优化 vs 强化学习

> 当前提交版口径：本文档对应 `cylinder_baseline.java`、`optuna_optimize.py`、`comsol_runner.py` 与 2026-05-05 生成的 150 组圆柱簇 trial 数据。早期中间结果已不再作为提交依据。

---

## 1. 赛题要求与方法选择

赛题鼓励将强化学习与多物理场仿真结合，但核心目标并不是限定某一种算法形式，而是要求形成具备“探索-评估-进化”能力的 AI 设计流程。对当前圆柱簇问题，我们选择 Optuna TPE 贝叶斯优化，原因是：

1. 设计变量是低维连续参数，适合样本高效的黑箱优化。
2. 单次评估需要 COMSOL 多物理场稳态求解、电压搜索和侵蚀循环，评估成本高。
3. 体积守恒可以解析消元，温度和寿命约束可以由仿真结果直接判断。
4. 随机森林代理模型和 SHAP 可以直接解释设计规律，便于写入技术报告。

因此，本项目在圆柱簇阶段采用“贝叶斯优化 + COMSOL 高保真评估 + 代理模型解释”的路线；zigzag 拓扑族仍作为更高维、更接近拓扑搜索的问题继续推进。

---

## 2. 圆柱簇问题特征

圆柱簇采用 8 段轴向分段，提交版搜索对称构型：

```text
[r1, r2, r3, r4, r4, r3, r2, r1]
```

体积守恒由解析关系保证：

```text
r1^2 + r2^2 + r3^2 + r4^2 = 4 * r0^2
r4 = sqrt(4*r0^2 - r1^2 - r2^2 - r3^2)
```

其中 `r0=2.5 mm`。Optuna 实际搜索 `r1,r2,r3` 三个自由变量，`r4` 由体积守恒推出。

| 特征 | 当前设置 | 对方法选择的影响 |
|---|---|---|
| 自由维度 | 3 个连续变量 | BO 在低维昂贵黑箱问题上效率高 |
| 体积约束 | 解析满足 | 避免无效 trial 和惩罚项调参 |
| 温度约束 | `Tmax < 3273.15 K` | 由 COMSOL 结果直接判断 |
| 寿命约束 | `lifetimeH >= 34.65 h` | 保留 baseline 30% 以上寿命 |
| 优化目标 | 最大化 `lifeAvgP03sphere_W` | 单目标，适合 TPE 采样 |
| 单次评估 | 分钟级 COMSOL 求解 | RL 所需样本数不可承受 |

---

## 3. 为什么当前阶段不采用 RL

强化学习适合序列决策问题，例如“在几何局部逐步增材/减材，再根据状态反馈调整下一步动作”。但圆柱簇阶段本质上是一次性参数设定：

- 状态不是高维场图，而是 `r1,r2,r3`。
- 动作不是序列修改，而是一次性给出半径组合。
- 奖励不是延迟反馈，而是一次 COMSOL 评估后的确定性指标。

在这种情况下，RL 会退化为样本效率较低的黑箱优化器。按常规经验，PPO/SAC 往往需要成千上万次 episode 才能形成稳定策略，而本项目圆柱簇的总预算只有约 150 次 COMSOL 评估。因此，BO 是更符合工程约束的选择。

## 4. BO 的实现细节

### 4.1 条件边界采样

`optuna_optimize.py` 中先采样 `r1`，再根据剩余平方半径预算动态计算 `r2` 和 `r3` 的可行上下界，最后解析得到 `r4`。这样可以保证：

- 每个进入 COMSOL 的 trial 都满足体积守恒。
- `r4` 不会落到最小半径以下。
- 搜索空间不会被大量无效样本浪费。

### 4.2 高保真评估

`comsol_runner.py` 复现 `cylinder_baseline.java` 的物理流程：

1. 从空白模型创建 8 段圆柱几何。
2. 建立导电介质、传热、焦耳热耦合和 `MultipleSpectralBands` S2S 面-面辐射。
3. 通过电压搜索找到满足 `Tmax < 3273.15 K` 的最大工作电压。
4. 在固定工作电压下执行侵蚀循环，任一段半径损失达到 20% 时判定失效。
5. 输出 `lifeAvgP03sphere_W`、寿命、温度、电功率和自遮挡诊断指标。

### 4.3 结果记录与续跑

- `data/trials.csv` 保存每个 trial 的半径、物理指标、状态和耗时。
- `data/optuna.db` 保存 Optuna study，可断点续跑。
- COMSOL 网格失败的异常 trial 记录为 `ERROR`，不参与最优筛选。

## 5. 当前 150 组结果

基线为均匀圆柱：

| 指标 | Baseline |
|---|---:|
| 半径 `[r1,r2,r3,r4]` | `[2.500, 2.500, 2.500, 2.500] mm` |
| `Vwork` | 0.6348 V |
| `initialTmax` | 3196.2 K |
| `lifetimeH` | 115.50 h |
| `lifeAvgP03sphere_W` | 428.57 W |

当前最优可行 trial 为 Trial 122：

| 指标 | Trial 122 |
|---|---:|
| 半径 `[r1,r2,r3,r4]` | `[1.592, 3.706, 1.737, 2.391] mm` |
| 对称 8 段构型 | `[1.592, 3.706, 1.737, 2.391, 2.391, 1.737, 3.706, 1.592] mm` |
| `Vwork` | 0.9766 V |
| `initialTmax` | 3270.0 K |
| `lifetimeH` | 35.27 h |
| `lifeAvgP03sphere_W` | 559.64 W |
| 相对 baseline 提升 | 30.58% |

该结果接近温度上限并贴近寿命硬约束下限，体现出优化器在“提高 0-3 μm 辐射功率”和“保留最低寿命”之间找到的边界解。

## 6. 可解释性分析

优化完成后运行：

```powershell
python parse_results.py
python surrogate_analysis.py
```

得到以下解释材料：

- `optimization_history.png`：显示 BO 搜索逐步接近高功率区域。
- `pareto_front.png`：展示辐射功率和寿命之间的权衡。
- `parallel_coordinate.png`：观察半径组合与性能指标的关系。
- `feature_importance.png`：随机森林给出的全局参数重要性。
- `shap_summary_p03.png` / `shap_summary_life.png`：分别解释 AvgP03 和寿命的主要驱动因素。
- `response_surface.png`：用代理模型展示设计空间响应面。

当前数据表现出的主要规律是：局部细颈可以提高电阻和工作温度，从而提高 0-3 μm 波段输出；较粗段则承担体积守恒和寿命缓冲作用。因此最优解倾向于“细-粗-细-中等”的对称分段结构，而不是均匀圆柱。

## 7. 方法结论

对圆柱簇阶段，Optuna TPE 贝叶斯优化比 RL 更合适：

| 维度 | BO/TPE | RL |
|---|---|---|
| 低维连续参数 | 适配 | 可用但样本效率低 |
| 昂贵 COMSOL 评估 | 样本效率高 | 训练预算不足 |
| 体积守恒约束 | 可解析内嵌 | 需投影或惩罚项 |
| 断点续跑 | SQLite 原生支持 | 需额外 checkpoint |
| 可解释性 | RF + SHAP 直接支持 | 需额外策略解释 |
| 当前项目成熟度 | 已跑完 150 组并生成图表 | 不适合作为圆柱簇主线 |

提交报告中建议表述为：本项目并非回避强化学习，而是根据问题维度、评估成本和约束结构，在圆柱簇低维参数阶段选择更样本高效的 BO；对于 zigzag 等更高维拓扑候选，再继续保留 RL/进化算法的扩展空间。
